## 项目结构

```text

├── base                    # 基础工具类
├── common                  # 公共模块
├── conf                    # 配置文件
├── data                    # 公共测试数据
├── backup                  # 数据库备份文件
├── logs                    # 日志文件
├── conf                    # 配置文件
├── report                  # 测试报告
│   ├── allureReport       # Allure报告
│   ├── temp               # 临时文件
│   └── tmreport           
├── testcase               # 测试用例
│   ├── Business Scenario  # 业务流程测试
│   └── Single             # 单接口测试
├── conftest.py            # pytest配置文件
├── environment.xml        # 环境配置
└── run.py                 # 启动入口
```
